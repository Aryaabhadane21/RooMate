# RooMate – Find Your Perfect Shared Home in India 

**RooMate** is a modern, interactive web application designed to simplify the process of finding shared accommodation and compatible flatmates across India's top cities. Built with a focus on user experience and lifestyle compatibility, it offers a seamless way to browse listings and connect with like-minded individuals.

---

## Key Features

### 1. Smart Room Listings
Browse verified room listings in cities like Mumbai, Bengaluru, Delhi, and Hyderabad.
- **Dynamic Filtering:** Search by location, budget, and configuration (2BHK, 3BHK, etc.).
- **Detailed Insights:** View per-person costs, deposits, and "Safety Scores" for Every listing.
- **Save for Later:** Bookmark your favorite rooms with instant notifications.

### 2. Swipe-to-Match (Find People)
Finding a roommate just got easier with our Tinder-style swipe interface.
- **Tinder UI:** Swipe right to like or left to pass on potential flatmates.
- **3D Compatibility Blueprint:** An interactive animation that compares your lifestyle traits (cleanliness, sleep schedule, etc.) against others to show matches and clashes.
- **Onboarding Journey:** A guided tour for new users to set up their search preferences.

### 3. Compatibility Quiz 
Not sure what you're looking for? Our quiz helps you define your lifestyle profile.
- **7-Question Deep Dive:** Covers sleep habits, guests, pets, and financial management.
- **Personalized Recommendations:** Get a compatibility score and a list of top-matched individuals.

### 4. Profile & Utility Calculator
Manage your preferences and plan your finances.
- **Utility Splitter:** Automatically calculate your share of electricity, internet, and other bills based on the number of roommates.
- **Trait Management:** Toggle your lifestyle tags to help others find you.
- **Live Chat Simulation:** Engage in mock conversations with potential matches through our floating chat system.

---

## Technology Stack

- **Frontend:** Semantic HTML5, Modern CSS3 (Grid, Flexbox, Custom Properties).
- **Interactivity:** Vanilla JavaScript (ES6+), Intersection Observer API for animations.
- **UX/UI:** Mobile-first responsive design, 2D/3D CSS animations, and interactive swipe gestures.

## Project Structure

- `listings.html` – Homepage / Main room search.
- `people.html` – Flatmate discovery and swipe interface.
- `quiz.html` – Lifestyle and compatibility assessment.
- `profile.html` – User account and budget tools.
- `/css/style.css` – Centralized design system.
- `/js/` – Modular logic for each page (swipe, quiz, filters).
